📞 Dashboard Televendas

Dashboard diário de televendas desenvolvido em HTML e JavaScript puro.


✨ Funcionalidades


Registro de valor vendido no dia com formatação automática em BRL
Controle de visitas feitas e efetivadas com cálculo de taxa de conversão
Gráfico de rosca com visualização das visitas convertidas
Cálculo automático de ticket médio por pedido
Painel de produtos vendidos com destaque para o mais vendido
Painel de input interativo para lançamento dos dados do dia
Salvamento dos dados no próprio HTML — gera um arquivo por dia com os dados embutidos

💡 Contexto

Desenvolvido para substituir o registro manual de resultados de televendas, centralizando em um único painel os principais indicadores do dia: vendas, visitas, conversão e produtos.


🚀 Como usar


Abra o arquivo index.html no navegador
Clique em ✏️ editar para abrir o painel de input
Preencha os dados do dia (data, venda, visitas, pedidos e produtos)
Clique em Calcular para atualizar o dashboard
Clique em Salvar HTML para gerar um arquivo do dia com os dados embutidos
